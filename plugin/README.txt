== Description ==
Connect your WordPress blog to Trafficontent to automatically receive and publish AI-generated blog posts based on your preferences and schedule. Perfect for store owners, content marketers, and SEO managers.

== Installation ==
1. Download the plugin as a `.zip` file.
2. In your WordPress admin panel, go to **Plugins â†’ Add New**.
3. Click **Upload Plugin**, then select the `.zip` file and install.
4. Activate the plugin.
5. Go to **Trafficontent** in your WordPress admin sidebar.
6. Paste your API key from your Trafficontent dashboard and click **Save API Key**.

== Frequently Asked Questions ==

= Where can I find my Trafficontent API key? =
Log into your account at https://trafficontent.com, go to your dashboard, and copy your API key under "Account Settings".

= Can I disable auto-publishing? =
Currently, all posts fetched from Trafficontent are published immediately. Future versions may offer a draft/publish toggle.
